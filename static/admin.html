添加 ChatGPT access_token

怎么获取 access_token？
1. 浏览器登录 chatgpt.com
2. 打开新标签访问 https://chatgpt.com/api/auth/session
3. 复制返回 JSON 里的 accessToken 字段（很长一段 eyJhbG... 开头）
4. 粘贴到下方文本框（多个 token 一行一个）
